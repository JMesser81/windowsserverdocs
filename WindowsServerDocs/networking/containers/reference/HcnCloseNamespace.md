# HcnCloseNamespace

## Syntax
`HRESULT result = HcnCloseNamespace(Namespace)`

### Parameters
|Parameter     |Description|
|---|---|---|---|---|---|---|---| 
|`Namespace`| **[In]** handle to a Namespace|
|    |    | 



## Return Values
|Return | Description|
|---|---|
|`S_OK`|On success|
|HResult error code|Failure|
|     |     |

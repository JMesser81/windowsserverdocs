# HcnCloseNetwork

## Syntax
`HRESULT result = HcnCloseNetwork(Network)`

### Parameters
|Parameter     |Description|
|---|---|---|---|---|---|---|---| 
|`Network`| **[In]** handle to a Network|
|    |    | 



## Return Values
|Return | Description|
|---|---|
|`S_OK`|On success|
|HResult error code|Failure|
|     |     |

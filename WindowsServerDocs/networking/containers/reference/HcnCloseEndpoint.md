# HcnCloseEndpoint

## Syntax
`HRESULT result = HcnCloseEndpoint(Endpoint)`

### Parameters
|Parameter     |Description|
|---|---|---|---|---|---|---|---| 
|`Endpoint`| **[In]** handle to a Endpoint|
|    |    | 



## Return Values
|Return | Description|
|---|---|
|`S_OK`|On success|
|HResult error code|Failure|
|     |     |
